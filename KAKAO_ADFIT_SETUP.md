# 카카오 애드핏 설정 가이드

## 필요한 정보
카카오 애드핏을 사용하려면 **광고 단위 ID (Unit ID)**가 필요합니다.

## 설정 방법

### 1단계: 카카오 애드핏 가입
1. https://adfit.kakao.com 접속
2. 카카오 계정으로 로그인
3. 회원가입 및 본인인증 완료

### 2단계: 매체 등록
1. 대시보드에서 "매체 등록" 클릭
2. 웹사이트 정보 입력:
   - 사이트 URL: `https://ssikapicktem.co.kr`
   - 사이트명: 씩아픽템
   - 카테고리 선택
3. 매체 승인 대기 (보통 1-3일 소요)

### 3단계: 광고 단위 생성
1. 승인된 매체 선택
2. "광고 단위 생성" 클릭
3. 광고 형식 선택:
   - 배너 광고 (가로형)
   - 네이티브 광고
   - 등
4. 광고 단위 이름 입력 (예: "메인 배너")
5. 광고 단위 생성 완료

### 4단계: 광고 단위 ID 확인
1. 생성된 광고 단위 클릭
2. **광고 단위 ID (Unit ID)** 복사
   - 형식: `DAN-xxxxxxxxxxxx` 또는 숫자로 된 ID

### 5단계: 코드에 적용
`AdRotator.tsx` 파일에서 `kakaoAdUnitId`에 발급받은 ID를 입력하세요:

```tsx
<AdRotator 
  type="auto"
  kakaoAdUnitId="발급받은_광고_단위_ID"
/>
```

또는 모든 AdRotator에 일괄 적용하려면:
`src/components/ads/AdRotator.tsx` 파일의 30번째 줄을 수정:

```tsx
kakaoAdUnitId = '발급받은_광고_단위_ID',  // 여기에 입력
```

## 주의사항
- 카카오 애드핏은 웹사이트에서도 사용 가능하지만, 주로 모바일 트래픽에 최적화되어 있습니다
- 광고 단위 ID는 매체별로 다르게 발급됩니다
- 광고 승인까지 시간이 걸릴 수 있습니다
