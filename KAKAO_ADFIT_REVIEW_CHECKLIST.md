# 카카오 애드핏 재심사 체크리스트

## 현재 상태
- ✅ 광고 코드 구현 완료
- ✅ 광고 단위 ID: `DAN-TuJyMLJV5hB5UXiO`
- ✅ 스크립트 자동 로드 설정 완료
- ⚠️ 심사 보류: 광고 설치 확인 필요

## 재심사 전 확인 사항

### 1. 배포 확인
- [ ] 사이트가 실제로 배포되어 있는지 확인 (`https://ssikapicktem.co.kr`)
- [ ] 배포된 사이트에서 광고가 표시되는지 확인

### 2. 광고 코드 확인
브라우저 개발자 도구(F12)에서 확인:
- [ ] `<ins class="kakao_ad_area">` 태그가 있는지 확인
- [ ] `data-ad-unit="DAN-TuJyMLJV5hB5UXiO"` 속성이 있는지 확인
- [ ] `//t1.daumcdn.net/kas/static/ba.min.js` 스크립트가 로드되는지 확인

### 3. 네트워크 확인
개발자 도구 > Network 탭에서:
- [ ] `ba.min.js` 파일이 성공적으로 로드되는지 확인
- [ ] 카카오 애드핏 관련 요청이 있는지 확인

### 4. 재심사 요청
1. 카카오 애드핏 대시보드 접속
2. 해당 광고 단위 선택
3. "재심사 요청" 버튼 클릭
4. 다음 정보 확인:
   - 사이트 URL: `https://ssikapicktem.co.kr`
   - 광고 코드가 실제로 설치되어 있음
   - 광고가 정상적으로 표시됨

## 문제 해결

### 광고가 표시되지 않는 경우
1. 브라우저 콘솔에서 에러 확인
2. 광고 차단 프로그램 비활성화
3. 캐시 삭제 후 재시도
4. 모바일 기기에서 확인 (카카오 애드핏은 모바일 최적화)

### 자동 감지가 안 되는 경우
- React 앱이므로 동적으로 생성되는 코드는 감지가 어려울 수 있음
- 실제 사이트에서 광고가 표시되는 스크린샷을 첨부하여 재심사 요청
- 개발자 도구에서 확인한 HTML 코드 스크린샷 첨부

## 참고
- 스크립트 설치 후 실제 광고 호출까지 최대 30분 소요될 수 있음
- 심사는 보통 1-3일 소요
- 모바일 환경에서 테스트 권장
